# nfcpy_id_reader

Read the card ID (Mifare: UID, FeliCa: IDm) with [nfcpy](https://github.com/nfcpy/nfcpy/) (a Python module).

## Requirement
### Python version
  - v3.x

### Operation System
  - Raspbian
  - macOS

## Install
```
pip install -U nfcpy-id-reader
```

## Usage
```
python -m nfcpy_id_reader
```
